# WordCount
# 如何运行
下载src文件到本地编译成功后，打开文件命令行，进入生成的exe文件所在目录，输入WordCount.exe 文件1 文件2即可运行
文件1文件2默认再同目录下，可以使用绝对地址运行其他位置的文件
（编译文件看这篇博文[Windows命令行编译.cpp程序](https://blog.csdn.net/u013767242/article/details/79748067))
# 功能简介
* 统计文件1的字符数，输出到文件2
* 统计文件1的单词数和有效行数，输出到文件2
* 统计文件1中最频繁的10个单词，分别输出到文件2
